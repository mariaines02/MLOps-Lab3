# MLOps-Lab2-demo

Demo for the second lab of the MLOps subject. In this lab you have to learn
* To create a Docker file to conteinarize a project.
* To create a GUI for the application (using Gradio) that will be hosted in HuggingFace spaces.
* To construct a CICD workflow (pipeline) using GitHub actions (.yml file).
